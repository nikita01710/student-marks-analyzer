# Student Marks Analyzer

This is a Python project that analyzes student marks from a CSV file using Pandas and NumPy.  
It calculates averages, pass/fail status, and displays graphs of marks.

## Features
- Reads marks from CSV
- Calculates average marks per student
- Shows pass/fail status
- Visualizes marks with bar charts

## How to Run
1. Make sure you have Python installed.
2. Install required libraries: pandas, numpy, matplotlib
3. Run the script using: `python student_marks_analyzer.py`
